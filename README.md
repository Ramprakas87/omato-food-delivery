# 🍔 OMATO Food Delivery Analytics Dashboard

## 📌 Project Overview

The **OMATO Food Delivery Analytics Dashboard** is a professional **Power BI Business Intelligence project** developed to analyze food delivery operations, customer behavior, payment trends, and delivery performance.

This project transforms raw transactional data into meaningful business insights using:

* Data Cleaning
* Data Transformation
* Data Modeling
* DAX Calculations
* Interactive Visualizations
* KPI Monitoring

The dashboard helps businesses monitor performance and make data-driven decisions efficiently.

---

# 🎯 Project Objectives

* Analyze food delivery transactions
* Monitor customer ordering behavior
* Track delivery performance
* Identify top-selling food items
* Understand payment preferences
* Generate actionable business insights
* Build an interactive dashboard for decision-making

## Development Process

### 1️⃣ Business Requirement Understanding

Understanding the business problem and defining dashboard goals.

### 2️⃣ Understanding the Dataset

Analyzing available data fields and identifying required metrics.

### 3️⃣ Data Connection

Importing dataset into Power BI from Excel/CSV sources.

### 4️⃣ Data Cleaning & Quality Check

Handling:

* Missing values
* Duplicate records
* Incorrect data types
* Data formatting issues

### 5️⃣ Data Modeling

Creating relationships between tables using a structured data model.

### 6️⃣ Data Processing

Transforming raw data into analysis-ready format using Power Query.

### 7️⃣ DAX Calculations

Creating:

* KPIs
* Measures
* Calculated Columns
* Aggregations

### 8️⃣ Dashboard Designing

Designing a clean, professional, and interactive layout.

### 9️⃣ Visualization & Formatting

Building charts, cards, slicers, and graphs for better storytelling.

### 🔟 Business Insight Generation

Extracting insights to support business decision-making.

# 📌 Business Insights

## 🔹 Food Performance Insights

* Samosa recorded the highest order quantity.
* Butter dishes and Chole Bhature showed strong demand.
* Biryani maintained stable sales performance.

## 🔹 Payment Insights

* UPI was the most preferred payment method.
* COD transactions remained significantly high.
* Card payments had lower adoption.

## 🔹 Delivery Insights

* More than 94% of orders were successfully delivered.
* Cancellation rate remained below 6%.

## 🔹 Monthly Trend Insights

* February achieved the highest transaction volume.
* April showed a decline in order quantity.
